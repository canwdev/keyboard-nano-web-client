# Keyboard nano 网页客户端

基于 `Vue 3 + TypeScript + Vite` 的 Keyboard nano 纯网页客户端。

打开网页即可使用，通过浏览器 `WebHID` 直接与设备通信，无需安装本地上位机，也不依赖额外后端服务。

## 主要功能

- 网页直连 Keyboard nano 设备
- 读取和保存设备设置、按键配置、LED 配置
- 支持灯光组颜色调整与前端灯效预览
- 保持与现有设备协议和桌面上位机功能对齐

更多开发说明见 [AGENTS](AGENTS.md)。
